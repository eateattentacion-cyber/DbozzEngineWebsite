Features

Everything you need to build games

A complete game development toolkit designed for flexibility and performance.

⚙

Entity Component System

Flexible ECS architecture for organizing game objects. Add, remove, and query components with a clean template-based API.

◆

PBR Renderer

OpenGL-powered physically based rendering pipeline with skybox support, real-time lighting, and shader-based materials.

⍈

Butsuri Physics

Custom physics engine with rigid body simulation, box and sphere colliders, gravity, and collision detection.

▶

Animation System

Skeletal animation with cross-fade blending, visual state machine editor, parameter-driven transitions, and SLERP interpolation.

✎

Full Editor

Professional dark-themed editor with hierarchy view, inspector, asset browser, scene save/load, undo/redo, and keyboard shortcuts.

≈

C# Scripting

Mono-powered C# scripting with a built-in script editor, syntax highlighting, and full API for controlling entities and components.

★

Scene Management

JSON-based scene serialization with project system, asset browser, and DabozzHub launcher for managing multiple projects.

⚙

PB&J Build System

Custom Python-based build system with incremental compilation, dependency tracking, parallel builds, and Qt MOC/RCC integration.

☰

Model Import

Import 3D models from OBJ, FBX, GLTF, GLB, DAE, and more via Assimp. Full support for meshes, textures, and skeletal data.

Get Started

Up and running in minutes

Clone, build, and start making games.

terminal
# Clone the repository
git clone https://github.com/eateattentacion-cyber/DbozzEngine.git
cd DbozzEngine

# Build the engine
python pbj.py build --target release

# Build the hub
cd dabozzhub
python pbj.py build --target release

# Launch
./bin/DabozzHub.exe
